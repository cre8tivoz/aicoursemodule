# Module 07 — Capstone build: ship your static mini-site

**Course outcomes:** CO-03, CO-04, CO-05

## Module objectives (AQF 5)

1. Execute a **staged** path from brief to **deployed** URL with **Lighthouse accessibility** evidence on the live site.
2. **Self-check** capstone rows (structure, accessibility, visual design, deployment, content) before submission.
3. Align personal **hosting** and **tooling** choices with **CO-05** (justify fit, cost, privacy).

## Prerequisites

- Modules 01–06 (especially M06 brief, prompts, review loop; M03 checklist; M05 semantics).
