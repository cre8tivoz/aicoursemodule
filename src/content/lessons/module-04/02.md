# Open-weight families — high level only

The **market moves monthly**. Exact model names and “best” leaderboards will change before this page is old. What you need is **orientation**:

- **Qwen** (Alibaba family) — often strong multilingual options; check **current** licence on the official site.  
- **DeepSeek** — often discussed for reasoning and coding; same rule: **read the licence** today.  
- **MiniMax-class** — vendor “mini” stacks and similarly named releases appear in leaderboards; treat **as a category**, not a single product you must use.

**Ollama**, **LM Studio**, **vLLM**, and similar tools are **runners**: they **load** weights and talk to your IDE or terminal. **You** still choose **which** weights and **whether** IT allows them.

## Do this — one bookmark (15 minutes)

1. Pick **one** path you might actually try later (examples: `ollama.ai`, `lmstudio.ai`, or your Linux package manager docs).  
2. Open the **official** “getting started” or install page.  
3. **Bookmark** it.  
4. In **one sentence**, write what the licence allows **you** to do (personal use? commercial? attribution?).  
5. If you cannot find a licence link, **stop**—that is a red flag.

> **Do not** download random weight files from forums this week. This course wants **habits**, not a GPU lab on day one.

## Concept — why we stay non-prescriptive

Hardware, drivers, and OS differ. Your **work machine** may block installs. A **Chromebook** may not support local inference at all. The professional skill is: **find the doc**, **read the licence**, **ask IT**—not memorise a blog post from last month.

## Deep dive — explore open-weight families (browser **or** on-device)

The high-level names above are not trivia—they are **entry points** you can **try** safely. This section encourages **exploration**: compare how different families answer the **same** prompt, then decide what is worth installing later.

> **Safety first**  
> Use **only** non-sensitive text (hobby copy, public tutorial snippets, fake “Acme Corp” examples). **Never** paste employer, client, or personal identity data into any chat or local tool without approval.

### Why both “browser chat” and “on device” matter

| Path | What you learn | Trade-off |
| ---- | ---------------- | --------- |
| **Browser chat** | Fast taste-test of a **vendor’s** model UX; no GPU setup | Traffic usually goes to **their** servers—check terms and privacy policy |
| **On device** (later) | True **local** inference when configured correctly; no per-message cloud bill for hobby use | Needs RAM/VRAM, time, and sometimes IT approval |

You are building **judgement**, not a single favourite brand.

### Browser — good starting points (verify URLs stay current)

These are **illustrative** entry points; products rename pages often. If a link fails, search the **official** vendor name plus “chat” or “console.”

**Quick links (official sites—navigate to Chat / Product from there):**

- [DeepSeek](https://www.deepseek.com/) — use their **Chat** product when available in your region.  
- [MiniMax](https://www.minimax.io/) — check **MiniMax 2.5** (or current) model pages; **free tiers** and naming change—read **today’s** terms.

1. **DeepSeek** — The vendor offers a **web chat** so you can test **DeepSeek** models without installing weights first. Try prompts such as: “Explain this HTML: `<nav><a href='/'>Home</a></nav>` in plain language.” Compare the answer to what you get from another chat you already use.  
2. **MiniMax** — **MiniMax** (including **MiniMax 2.5**-class releases) is often available through the vendor’s **web app** or developer console. At **time of writing**, vendors sometimes offer **free** access for exploration—**verify** the live **pricing** and **acceptable use** pages before you rely on it for coursework. If you see **2.5** in the name, that is a **version line**, not a guarantee the same offer stays free.  
3. **Qwen** — Explore **Alibaba Cloud** / **Qwen** official pages and any **playground** or **Hugging Face** model cards linked from the official ecosystem. Good prompt: “Give me a minimal accessible `header` / `main` / `footer` skeleton in HTML5.”  
4. **Hugging Face** — [huggingface.co](https://huggingface.co/) hosts **model cards**, **Spaces** (browser demos), and discussion—useful for **comparing** open-weight releases **without** installing yet.

**Do this (20 minutes):** Open **two** browser experiences (e.g. DeepSeek Chat **and** MiniMax **or** Qwen playground). Ask the **same** three prompts (HTML explain, rewrite paragraph in plain language, suggest a nav structure). **Note** differences: tone, length, hallucination risk, and whether they cite uncertainty.

### On device — when you are ready (after you know your RAM)

**Runners** load **weights** you choose:

- **[Ollama](https://ollama.com/)** — popular CLI/app; `ollama pull <model>` then run locally **if** your machine can handle the tag you pick.  
- **[LM Studio](https://lmstudio.ai/)** — desktop UI to search, download, and chat locally.  
- **vLLM**, **llama.cpp**, and others — advanced; skip until you need them.

**Do this (optional, 30+ minutes):** If IT and hardware allow, install **one** runner from the **official** site only. Pull **one small** model tag recommended for “chat” on CPU (read the runner’s docs—names change weekly). Ask: “Summarise `index.html` in my practice folder” **after** you open the folder in the tool’s allowed scope.

### How this ties to your static site course

- **Browser exploration** trains **prompting** and **comparison**—the same skills you use in an IDE with extensions.  
- **On-device exploration** trains **proportionality** (Lesson 03): is this model **worth** the disk and heat for a **one-page** hobby site?

Pricing and model version numbers change — read current terms at the source. This course does not endorse a single vendor.

## Key takeaways

- **Families** = map reading; **versions** = temporary detail.  
- **Runners** (Ollama, LM Studio, …) are tools; **models** are separate choices.  
- **Browser chats** (e.g. DeepSeek, MiniMax web) are a **low-friction** way to compare families before you install anything—always with **non-sensitive** prompts.
